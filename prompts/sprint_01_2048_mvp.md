## 템플릿 1. 프로젝트 초기 세팅

```md
새 프로젝트를 시작한다.
프로젝트는 2048 퍼즐 + 커피 매장 SNG/타이쿤 결합 게임이다.

핵심 방향:
- 마이크로인터랙션이 매우 중요
- 몰랑하고 부드러운 촉감
- 세련되고 고급스러운 코지 감성
- 퍼즐과 매장 운영이 하나의 루프로 연결되어야 함

기술 스택:
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand

이번 작업 목표:
- 프로젝트 기본 구조 생성
- 추천 폴더 구조 반영
- 공통 디자인 토큰 기초 세팅
- 기본 라우트 생성 (로비, 퍼즐, 업그레이드)
- 공통 UI 베이스 컴포넌트 생성

수정/생성 범위:
- app/*
- components/common/*
- features/*
- stores/*
- lib/*
- docs/*
- prompts/*

요구사항:
- 구조를 과하게 만들지 말고 이후 확장이 쉬운 형태로
- 공통 버튼, 카드, 모달, 자원칩 정도는 먼저 준비
- 디자인 톤은 따뜻한 크림/브라운/우드 계열
- 전체적으로 카드형 UI와 부드러운 모션 기반

완료 후 아래 형식으로 보고:
1. 작업 요약
2. 생성/변경 파일 목록
3. 구조 설명
4. 실행 방법
5. 다음 추천 작업
```

## 운영 팁

- 작업은 항상 작게 자른다.
- 기능 작업과 polish 작업을 섞지 않는다.
- docs / prompts 업데이트를 자주 시킨다.
- 한 세션에 한 목표만 명확히 둔다.
- “왜 이 작업을 하는지”를 항상 같이 적는다.
