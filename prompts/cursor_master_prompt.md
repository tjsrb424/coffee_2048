# Cursor Master Prompt

이 프로젝트는 2048 퍼즐 + 커피 매장 SNG/타이쿤 결합 게임이다.

절대 잊지 말아야 할 핵심:
- 마이크로인터랙션이 핵심 경쟁력이다.
- 터치, 클릭, 화면전환, 수치 변화 모두 부드럽고 세련되어야 한다.
- 몰랑몰랑한 촉감과 힐링 감성이 중요하다.
- 퍼즐과 매장 운영은 분리된 미니게임이 아니라 하나의 루프여야 한다.
- 기능 추가보다 손맛/반응감/가독성을 우선한다.

기술 스택:
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand

작업 원칙:
- 항상 작업 범위를 작게 나눈다.
- 수정 파일 범위를 명시한다.
- 구조를 먼저 보고, 필요한 최소 수정만 한다.
- 엔진 로직과 UI 컴포넌트를 분리한다.
- 보고 형식은 반드시 다음을 포함한다:
  1. 작업 요약
  2. 변경 파일 목록
  3. 구현 내용
  4. 테스트 방법
  5. 남은 이슈
  6. 다음 추천 작업

디자인 키워드:
- cozy
- premium
- soft
- tactile
- warm
- clean
- low-noise
- alive but not flashy
