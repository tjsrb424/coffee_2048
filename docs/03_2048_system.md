# 03. 2048 SYSTEM

## 목표

2048 파트를 단순 복제하지 않고, 이 게임만의 촉감과 메타 연결성을 갖는 핵심 엔진으로 만든다.

---

## 기본 보드 규칙

- 보드 크기: 4x4
- 조작: 스와이프 / 키보드 방향키(웹 테스트용)
- 기본 룰:
  - 같은 단계의 타일 2개가 만나면 1단계 상위 타일로 합체
  - 한 턴에 한 번만 합체
  - 이동 후 빈칸에 새 타일 생성
  - 이동 불가 상태면 세션 종료

---

## 타일 해석 방식

겉보기는 숫자보다 **커피 진화 체인**으로 보여준다.
내부 로직은 2, 4, 8, 16... 구조를 유지해도 된다.

### 예시 체인
- 2 = 생두
- 4 = 로스팅 원두
- 8 = 분쇄 원두
- 16 = 에스프레소 샷
- 32 = 아메리카노
- 64 = 라떼
- 128 = 바닐라 라떼
- 256 = 시그니처 블렌드
- 512 = 프리미엄 세트
- 1024 = 카페 스타 메뉴
- 2048 = 오늘의 마스터피스

초기엔 6~8단계만 준비하고 나중에 확장해도 된다.

---

## 세션 목표 구조

기본 점수만 주면 2048의 반복성이 빨리 온다.
따라서 세션마다 소목표를 붙인다.

### 목표 예시
- 로스팅 원두 타일 5회 합체
- 라떼 이상 타일 2개 생성
- 10콤보 달성
- 한 판 내 특수 타일 1회 사용
- 주문 메뉴 재료 충족

---

## 퍼즐 보상

### 기본 보상
- 점수
- 코인
- 원두 자원
- 메뉴 제작 재료

### 고급 보상
- 희귀 원두 조각
- 당일 판매 버프
- 고객 만족도 버프
- 속도 부스터
- 자동 로스팅 티켓

---

## 특수 타일 시스템

초기 버전부터 1~2종은 있어야 손맛이 살아난다.

### 1. 밀크폼 타일
- 주변 1칸에 부드러운 연쇄 보너스
- 합체 시 작은 범위 점수 보너스

### 2. 골든빈 타일
- 합체 시 코인 보너스
- 당일 매장 수익 버프 소량 획득

### 3. 바리스타 컵 타일
- 원하는 방향 추가 이동 또는 인접 타일 승급

### 4. 설탕 시럽 타일
- 일정 횟수 내 콤보를 잇게 돕는 보조 타일

MVP에서는 **골든빈 1종만** 먼저 넣어도 충분하다.

---

## 콤보 시스템

2048은 기본적으로 콤보 감각이 약하다.
이 프로젝트에서는 감성/타격감을 위해 **연쇄 피드백 시스템**을 별도로 강화한다.

### 콤보 정의
- 한 번의 스와이프에서 여러 합체가 연속 발생하면 콤보
- 연속 턴에서 특정 조건으로 합체 유지 시 체인 보너스

### 콤보 보상
- 추가 점수
- 자원 추가 드롭
- 사운드 단계 상승
- 화면 피드백 강화
- 김 연출 / 파티클 증가

### 연출 예시
- 2연쇄: 작은 팡
- 3연쇄: 화면 미세 진동 + 컵 링 효과
- 4연쇄 이상: 은은한 슬로모/확대/반짝임

---

## 난이도 곡선

### 초반
- 실수해도 세션이 금방 안 망해야 한다.
- 새 타일 분포가 안정적이어야 한다.
- 상위 타일 도달 경험을 빨리 줘야 한다.

### 중반
- 목표 플레이 유도
- 보드 공간 관리의 중요성 증가
- 특수 타일 활용

### 후반
- 주문/이벤트 목적에 맞는 타일 운영
- 희귀 보상 노리기
- 매장 수익 최적화

---

## 난이도 보정 장치

힐링 감성을 위해 너무 가혹하면 안 된다.

### 보정안
- 세션 실패 시 일부 재화 보존
- 광고/재화 없이도 기본 재시도 가능
- 하루 첫 3판 정도는 보정된 쉬운 분포
- 연속 실패 시 구제 아이템 소량 제공
- 특수 타일 등장 확률이 약간 증가

---

## 컨트롤 감성 설계

### 입력 반응
- 스와이프 판정은 관대하게
- 짧은 거리도 방향만 명확하면 인식
- 오입력 방지를 위한 임계값 필요

### 입력 후 반응 속도
- 입력 즉시 1프레임 내 방향 피드백
- 실제 이동은 짧고 매끈하게
- 완료 후 합체 반응이 명확히 보여야 함

### 피해야 할 것
- 입력했는데 잠깐 멈춘 뒤 이동
- 이동은 빠른데 합체가 밋밋함
- 파티클이 많은데 물성이 없음

---

## 세션 내 연출 층위

한 번의 이동에서 최소 5단계 피드백이 있을 수 있다.

1. 입력 인지
2. 타일 이동
3. 충돌/합체
4. 스코어/자원 획득
5. 보드 안정화 후 새 타일 생성

각 단계가 분리되어 보여야 손맛이 살아난다.

---

## 사운드 설계 방향

### 타일 이동
- 아주 작은 미끄러짐/톡 소리

### 합체
- 낮고 둥근 팝
- 단계가 높을수록 조금 더 묵직한 소리

### 고단계 합체
- 유리/세라믹 같은 맑은 잔향
- 과하지 않은 하이라이트

### 콤보
- 짧은 멜로디 상승

---

## 햅틱 설계 방향

모바일/앱 패키징 시 중요하다.

- 이동: 거의 없음 또는 매우 약함
- 일반 합체: 짧은 탭
- 상위 합체: 중간 탭
- 4연쇄 이상: 짧은 이중 탭

웹 MVP에서는 추상화 레이어만 만들어 두고, 이후 모바일 빌드에서 연결한다.

---

## MVP 규칙 범위

초기에는 아래만 구현한다.

- 4x4 보드
- 기본 합체
- 점수
- 1종 특수 타일
- 1종 세션 목표
- 세션 종료
- 자원 보상 반영

그 이후 확장한다.

---

## 확장 규칙 후보

### 1. 주문 카드
이번 판에서 특정 타일을 만들면 보너스

### 2. 날씨 시스템
비 오는 날은 따뜻한 메뉴 수요 증가
→ 특정 타일 보너스

### 3. 테마 보드
벚꽃 시즌, 겨울 카페, 밤 카페 등
시각 테마와 보상 테이블 변경

### 4. 바리스타 스킬
세션 중 1회 사용 가능한 스킬
- 셔플
- 한 칸 되돌리기
- 임의 승급
- 생성 타일 제어

---

## 가장 중요한 구현 포인트

이 퍼즐은 “기능적으로 맞는 2048”만으로는 실패다.
반드시 아래를 만족해야 한다.

- 입력 즉시 반응한다.
- 합체 순간이 기분 좋다.
- 합체 결과가 매장 성장과 연결된다.
- 실패해도 다시 만지고 싶다.
