# 02. CORE LOOP AND META

## 전체 루프 개요

이 게임은 크게 두 레이어가 맞물린다.

1. **퍼즐 레이어**
   - 2048 플레이
   - 합체
   - 점수/콤보/특수 타일 획득
   - 원두/재료/부스터 생성

2. **매장 레이어**
   - 원두 볶기
   - 메뉴 생산
   - 진열/판매
   - 매장 업그레이드
   - 고객 대응
   - 꾸미기/확장

핵심은 이 둘이 끊기지 않고 순환해야 한다.

---

## 분 단위 루프

### 1~3분 루프
- 2048 한 판 플레이
- 재화/재료 획득
- 짧은 보상 연출
- 매장 복귀
- 생산/진열/업그레이드

이 짧은 루프가 가장 중요하다.

---

## 세션 루프

### 5~10분 루프
- 오늘의 주문/목표 확인
- 2048 플레이로 재료 확보
- 메뉴 생산
- 진열/판매
- 일일 목표 달성
- 업그레이드 또는 인테리어 반영

---

## 중기 메타 루프

### 1~3일 루프
- 신메뉴 해금
- 머신 업그레이드
- 로스팅 등급 상승
- 고객층 확대
- 매장 테마 확장
- 새로운 퍼즐 규칙 또는 타일 테마 해금

---

## 장기 메타 루프

### 1~4주 루프
- 매장 확장
- 지역 이동
- 시그니처 메뉴 수집
- 이벤트 시즌 참여
- 한정 인테리어 획득
- 새로운 고객 계층과 특별 주문 해금

---

## 권장 플레이 흐름

### 흐름 A. 기본
1. 로비 진입
2. 현재 재고/주문/고객 수요 확인
3. 2048 진입
4. 원두/재료/부스터 획득
5. 로비 복귀
6. 원두 볶기/음료 생산
7. 진열
8. 고객 판매
9. 매출/경험치/업그레이드
10. 다음 목표 확인

### 흐름 B. 하드코어 유저
1. 퍼즐에서 콤보를 최적화
2. 특정 상위 타일을 노려 희귀 재료 생산
3. 매장 생산 라인을 최적화
4. 주문 효율/회전율을 올림
5. 하루 매출 극대화

### 흐름 C. 코지 유저
1. 퍼즐을 편하게 플레이
2. 적당히 재고를 채움
3. 매장 분위기, 꾸미기, 단골 반응을 즐김
4. 천천히 성장

이 두 사용자층이 모두 즐길 수 있어야 한다.

---

## 퍼즐과 메타의 연결 규칙

퍼즐 결과가 아래 중 최소 3가지와 동시에 연결되어야 한다.

- 원두 획득
- 재료 획득
- 메뉴 경험치
- 당일 판매 버프
- 고객 만족도 버프
- 로스팅 부스터
- 희귀 메뉴 재료
- 이벤트 토큰

---

## 추천 연결 모델

### 모델 A. 직접 생산형
퍼즐에서 합체하면 곧바로 원두/재료를 얻는다.

장점:
- 직관적
- 이해가 쉬움
- 초반 진입장벽 낮음

단점:
- 운영과 생산이 너무 단순해질 수 있음

### 모델 B. 주문 충족형
퍼즐이 고객 주문 요구를 채워준다.
예: “오늘 라떼 6잔 분량 재료 확보”

장점:
- 목적이 생김
- 퍼즐 플레이가 더 전략적이 됨

단점:
- 초반에는 복잡하게 느껴질 수 있음

### 모델 C. 시간 진행형
퍼즐 한 번 움직일 때마다 매장 시간이 흐르고 고객이 움직인다.

장점:
- 퍼즐과 매장이 살아서 연결됨
- 몰입감 높음

단점:
- 설계와 튜닝 난이도 높음

### 권장안
초기 버전은 **A + 약한 B**로 간다.
즉,
- 퍼즐이 직접 자원을 준다.
- 동시에 오늘의 주문/목표가 있어 특정 타일을 유도한다.

## 하루 구조 예시

### 아침
- 오늘의 추천 메뉴 공개
- 날씨/기분 버프로 특정 메뉴 수요 증가

### 낮
- 퍼즐 세션 중심
- 고객 피크타임
- 재고 부족 시 추가 플레이 유도

### 저녁
- 남은 재고 정리
- 일일 정산
- 보너스 손님
- 업그레이드/장식 반영

모바일/캐주얼 특성상 실제 시간 흐름보다 “테마 구조”로만 구현해도 된다.

## MVP용 핵심 루프

초기 구현에서 반드시 들어가야 하는 최소 루프는 다음과 같다.

1. 퍼즐 플레이
2. 원두 획득
3. 로스팅
4. 커피 제작
5. 진열
6. 자동 판매
7. 코인 획득
8. 업그레이드
9. 다음 퍼즐 진입

이 루프만으로도 재미가 생겨야 한다.

## 플레이 동기 장치

### 짧은 동기
- 다음 합체 보고 싶다
- 지금 콤보 더 잇고 싶다
- 오늘 주문 하나만 더 채우고 싶다

### 중간 동기
- 메뉴 하나 해금하고 싶다
- 에스프레소 머신 업그레이드하고 싶다
- 매장 테마를 바꾸고 싶다

### 장기 동기
- 시그니처 카페로 성장하고 싶다
- 새로운 지역 매장을 열고 싶다
- 희귀 고객/이벤트를 해금하고 싶다

## 핵심 리스크

### 리스크 1. 2048만 재밌고 매장은 귀찮다
해결:
- 매장 조작을 최소화하고 결과가 시원하게 보이게 한다.
- 자동 판매/자동 재배치 등 편의 시스템을 초반부터 준다.

### 리스크 2. 매장은 예쁜데 퍼즐과 상관없다
해결:
- 퍼즐 없이는 생산 효율이 제한되도록 만든다.
- 당일 수요/주문이 퍼즐 목표를 계속 만든다.

### 리스크 3. 두 게임이 모두 애매하다
해결:
- 초반은 2048 손맛에 집중
- 운영은 단순하지만 보기 좋게
- 확장은 점진적으로

## 추천 MVP KPI 감각

정식 KPI가 아니라 설계 감각 기준이다.

- 첫 세션 10분 내:
  - 유저가 메뉴 1종 이상 생산
  - 업그레이드 2회 이상
  - 퍼즐 진입 3회 이상

- 1일차:
  - 메뉴 3종 해금
  - 고객 타입 2종 인지
  - 로비에 애착 형성

- 3일차:
  - 퍼즐에서 특정 상위 타일 목표 플레이 시작
  - 운영 효율 최적화 동기 발생
