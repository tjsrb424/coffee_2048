# Coffee 2048 × SNG 프로젝트 문서 세트

이 폴더는 **2048 퍼즐 + 커피 매장 SNG/타이쿤** 결합 게임을 **Cursor AI 중심으로 개발**하기 위한 기획/개발 문서 세트입니다.

## 문서 목록

- `01_GAME_VISION.md`
  - 게임 비전, 타깃 감성, 핵심 가치
- `02_CORE_LOOP_AND_META.md`
  - 핵심 루프, 메타 루프, 플레이 구조
- `03_2048_SYSTEM.md`
  - 2048 퍼즐 규칙, 스킬, 타격감, 리텐션 설계
- `04_CAFE_TYCOON_SYSTEM.md`
  - 로비/매장 운영/생산/진열/판매 시스템
- `05_UX_MOTION_DIRECTION.md`
  - 마이크로인터랙션, 애니메이션, 촉감 설계
- `06_CONTENT_ECONOMY_BM.md`
  - 성장 구조, 재화, 업그레이드, BM 방향
- `07_TECH_ARCHITECTURE.md`
  - 추천 기술 스택, 상태 구조, 데이터 구조, 폴더 구성
- `08_DEV_ROADMAP.md`
  - 실제 구현 순서, 스프린트 분리, 우선순위
- `09_CURSOR_WORKFLOW.md`
  - Cursor에게 어떻게 시킬지, 대화 분리법, 작업 운영법
- `10_CURSOR_TASK_TEMPLATES.md`
  - 바로 붙여넣을 수 있는 Cursor 전달문 템플릿

## 권장 개발 방식

이 프로젝트는 두 게임을 억지로 붙이는 방식이 아니라, 아래처럼 **한 게임의 두 레이어**로 설계해야 합니다.

1. **즉각적 재미 레이어**
   - 2048 퍼즐의 손맛
   - 스와이프, 합체, 폭발, 반응음, 진동 같은 즉각 피드백

2. **지속 동기 레이어**
   - 커피 매장 운영
   - 재료 생산, 진열, 판매, 인테리어, 성장, 지역 확장

핵심은 **2048을 잘하면 매장이 더 잘 돈다**는 연결입니다.

## 가장 중요한 방향

이 게임은 퍼즐과 경영을 동시에 넣는 만큼, 아래 3가지를 계속 지켜야 합니다.

- 퍼즐이 메타를 먹여 살려야 한다.
- 매장 메타가 퍼즐에 목표를 줘야 한다.
- 모든 상호작용은 “몰랑하고 세련된 힐링 감성”으로 통일되어야 한다.

## 첫 구현 추천 범위

처음부터 모든 시스템을 넣지 말고, 아래만 먼저 구현하는 것을 권장합니다.

- 2048 보드 4x4
- 합체 애니메이션 / 점수 / 콤보 / 진동 / 사운드
- 로비 1개
- 커피 원두 생산 1라인
- 메뉴 3종
- 자동 판매
- 업그레이드 5~8개
- 고객 3타입
- 저장/불러오기
- 하루/세션 진행

그 다음에야 아래를 확장합니다.

- 주문 시스템
- 로스팅/진열/재고 분리
- 인테리어
- 스태프
- 지역 확장
- 이벤트
- 시즌/데일리 미션
- 꾸미기 / 테마 / 스킨
